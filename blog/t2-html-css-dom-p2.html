  • What is grid based design?

    A grid based design is a way to organize the layout of the website. By creating rows and coloumns, one can easily arrange and move elements around a website.

  • What is all the hype about responsive webpages?

    Responsive webpages can be viewed easily on any size of screen. The webpage will adjust texts, images, menus and whatever else it needs to, as the size of the viewing platform changes.

  • What is semantic structure?

    The way we organize different semantic tags within other elements.
